# Cloning
Cloning my project? sure..?
- Requirements
  - .NET 7.0 SDK

To build the app just run the `BuildAPP.exe` if you have dotnet installed<br>
or run MSBUILD `msbuild Build/BuildLauncherDL.csproj`
